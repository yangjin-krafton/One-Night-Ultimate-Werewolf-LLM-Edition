# TTS 역할별 성격·말투 표 - rust_orbit

이 문서는 기존 [TTS 역할별 성격·말투 표](/mnt/d/weeks/one-night-ultimate-werewolf-llm-edition/docs/TTS%20%EC%97%AD%ED%95%A0%EB%B3%84%20%EC%84%B1%EA%B2%A9%C2%B7%EB%A7%90%ED%88%AC%20%ED%91%9C.md)를 그대로 두고, `rust_orbit` 전용으로 따로 쓰는 설정표다.

이번 문서의 핵심 규칙은 하나다.

- **원래 역할 이름은 그대로 유지한다.**
- 대신 호출할 때만 앞뒤로 군인 / 우주연구원 / 정거장 승무원 컨셉을 붙인다.
- 예: `늑대인간 대위`, `종자 이등병`, `예언자 박사`, `연구원 마녀`

## rust_orbit 톤 원칙

- 배경은 `녹슨 궤도 우주정거장`이다. 군 보안구역, 연구동, 격리실, 동면창, 기압문, 관제실, 정비 베이 같은 공간이 떠오르게 쓴다.
- 역할 이름 자체는 바꾸지 않는다. `늑대인간`, `종자`, `예언자` 같은 원래 명칭을 그대로 유지한다.
- 호칭은 `대화할 때 입에 붙는 순서`를 기본으로 한다.
- 군 계급은 보통 `역할 이름 + 계급`이 자연스럽다. 예: `늑대인간 대위`, `종자 이등병`
- 연구/실무 직책은 보통 `직책 + 역할 이름`이 더 자연스럽다. 예: `연구원 마녀`, `분석관 도플갱어`, `기록관 계시자`
- 군인 쪽은 `대위`, `중위`, `소위`, `상사`, `병장`, `이등병` 같은 계급을 쓴다.
- 연구원 쪽은 `박사`, `선임연구원`, `수석연구원`, `연구주임`, `분석관` 같은 직책을 쓴다.
- 정거장 진행자는 지나치게 감정적이지 않고, `브리핑`, `로그`, `격리`, `대상`, `확인`, `절차` 같은 단어를 자주 쓴다.
- 말투는 SF 군사 브리핑 느낌이지만, 플레이어가 즉시 이해할 수 있게 짧고 또렷하게 유지한다.
- 군인 역할은 `-요`를 쓰지 않는다. `-하십시오`, `-한다`, `-하라`, `-유지`, `-대기`, `-복귀` 같은 딱딱한 명령형/보고형으로 유지한다.

## 호칭 규칙

| 규칙 | 예시 |
|------|------|
| 역할 이름을 지우지 않는다 | `늑대인간 대위`, `강도 병장`, `연구원 마녀` |
| 대화체로 자연스러운 순서를 우선한다 | `도플갱어 장관`은 가능하지만 `도플갱어 분석관`보다 입에 붙는지 먼저 본다 |
| 군 계급은 뒤에 붙이는 편이 자연스럽다 | `늑대인간 대위`, `종자 이등병`, `강도 병장` |
| 연구/직책은 앞에 붙이는 편이 자연스럽다 | `연구원 마녀`, `분석관 도플갱어`, `기록관 계시자` |
| 같은 계열 역할은 호칭 체계를 맞춘다 | 늑대 팀은 군 계급, 정보직은 연구직, 교란직은 실무직 |

## 말끝 축

| 말끝 축 | 느낌 | 권장 역할 |
|------|------|------|
| `-하십시오 / -하면 됩니다 / -보고하십시오` | 군 브리핑, 절차 지시 | Narrator, werewolf, alpha_wolf |
| `-하시죠 / -보시죠 / -기록해 두시죠` | 연구실 브리핑, 분석형 안내 | seer, witch, aura_seer, revealer |
| `-해 / -하지 / -하자` | 실무직, 능청, 현장감 | robber, troublemaker, village_idiot, insomniac |
| `-유지하십시오 / -대기하십시오 / -종료합니다` | 보안/관제 멘트 | squire, beholder, minion, dream_wolf |
| `-관측된다 / -식별된다 / -기록한다` | 군 정보분석, 상황판단 | mystic_wolf |
| `-거든요 / -네요 / -보여요` | 관찰형, 설명형 구어체 | paranormal_investigator |

## 군인 말투 규칙

| 규칙 | 예시 |
|------|------|
| 군인 역할에는 `-요`를 쓰지 않는다 | `눈 떠 주세요` 대신 `눈 뜨십시오`, `대기하세요` 대신 `대기하십시오` |
| 문장은 짧게 끊는다 | `대상 확인. 은폐 유지. 복귀.` |
| 감탄보다 지시를 우선한다 | `좋네요` 대신 `확인. 이상 없음.` |
| 군 계급 호칭 뒤에는 바로 명령을 붙인다 | `늑대인간 대위, 대상 확인하십시오.` |
| 하위 계급일수록 보고체/복명체를 쓴다 | `확인했습니다`, `대기 중입니다`, `순서 유지하겠습니다` |

## 빠른 작성 팁

| 상황 | 권장 방식 |
|------|------|
| 호칭이 너무 자유롭게 섞일 때 | 반드시 `원래 역할 이름은 유지`하고, 대화체로 자연스러운 쪽을 고른다. 군 계급은 `늑대인간 대위`, 연구직은 `연구원 마녀`, `기록관 계시자`처럼 쓴다. |
| 늑대 팀이 모두 비슷해질 때 | `werewolf`는 보안대위, `alpha_wolf`는 사령관, `mystic_wolf`는 심리전장교, `dream_wolf`는 동면요원으로 갈라 쓴다. |
| 정보직이 전부 연구원처럼만 들릴 때 | `seer`는 박사, `apprentice_seer`는 인턴, `aura_seer`는 분석관, `paranormal_investigator`는 조사관, `beholder`는 감시관으로 나눈다. |
| 실무직이 너무 가벼워질 때 | `robber`는 병장, `troublemaker`는 하사, `village_idiot`는 이병, `insomniac`는 정비주임으로 계급 차를 분명히 둔다. |
| 우주정거장 톤이 약할 때 | `격리`, `로그`, `관제`, `대상`, `기록`, `잔향`, `교란`, `복귀`, `절차` 같은 단어를 넣는다. |

## 권장 템플릿

```text
[정거장 분위기 1문장]. [원래 역할 이름을 유지한 대화체 호칭], [눈 떠 표현]. [행동 핵심]. [필요하면 군/연구 추임새 0~1회]. [눈 감기 표현].
```

예시:

```text
관제실 로그에 미세한 흔들림이 잡혔습니다. 연구원 마녀, 눈 떠 보시죠. 중앙 샘플 하나만 읽어 보시죠. 데이터는 충분합니다. 실험 로그만 남기고 감으시죠.
```
