# 역할 동물화 비주얼 제안서

## 배경

현재 `scripts/role_portraits_prompts.json`에서 **늑대 진영**(`werewolf`, `alpha_wolf`, `mystic_wolf`, `dream_wolf`)과 **반인반수**(`squire`의 늑대 귀, `minion`의 늑대 송곳니)만 짐승 요소를 갖고 있고, 나머지 21개 역할은 순수 인간형 비주얼이다.

결과적으로 역할 초상화를 한 판에 깔아 놓으면 **늑대 쪽만 우화·설화 톤이고 나머지는 다이쇼 시대극 초상**이 되어, 화풍은 같은데도 세계관 층위가 어긋나는 위화감이 생긴다.

본 문서는 **마을 진영·중립 진영까지 동물 수인화**해 27개 초상이 한 계열의 요괴·영수 도감처럼 읽히도록 하는 제안서다. `TTS 역할별 비주얼·보이스 매칭 표.md`에서 이미 정의된 **성별·연령·성격 축을 바꾸지 않는다**는 전제를 최상위 제약으로 둔다.

## 동물화 원칙

1. **성별·연령 유지**: 기존 매칭 표의 성별·연령 축(노년 남성 / 청년 여성 / 소녀 / 요정형 등)을 그대로 보존한다. 즉 동물만 바꾸고 "노년 여성 늑대(dream_wolf)"처럼 연령·성별이 함께 읽히게 털 색·주름·체격으로 표현한다.
2. **동아시아 요괴 레이어 우선**: Taisho Roman 스타일과 호응하는 일본·한국 민속 동물(키츠네, 타누키, 바케네코, 야타가라스, 학, 토끼, 거북)을 1순위로 배정한다. 서양 판타지 동물(유니콘, 그리핀)은 사용하지 않는다.
3. **기능·성격과 동물 상징 일치**: 사냥꾼은 맹금, 예언자는 부엉이처럼 **역할의 아이콘성과 동물의 상징성**이 한 번에 읽혀야 한다. 단순히 귀엽거나 미려한 동물을 배정하지 않는다.
4. **수인 비율은 "수인" 기준**: 완전 동물도, 사람 얼굴에 귀만 달린 형태도 아닌 **의인화된 반수**(털 덮인 얼굴 + 사람 체형 + 다이쇼 의상)로 통일한다. 기존 늑대 계열(4종)과 같은 톤이다.
5. **진영 단서 색상**: 늑대 진영은 회·적 계열 털, 마을 진영은 흰·갈·파스텔 계열 털, 중립/저주 계열은 검·자·흑적 계열 털로 **털 색으로 진영 힌트**를 준다.

## 기존 TTS 호칭·톤과의 호응

`TTS 역할별 비주얼·보이스 매칭 표.md` 하단 어긋남 체크리스트를 동물화 후 재확인해야 한다. 동물로 바뀌어도 **성별·연령 축은 유지**하므로 체크리스트의 교정안(`공작 부인`, `대모님`, `원로 전하` 등)은 그대로 유효하고, 아래 항목만 **동물 단어를 얹어 톤을 강화**할 여지가 있다.

| id | 기존 교정안 | 동물화 후 확장안 |
|------|------|------|
| `seer` | (호칭 유지) | `부엉이 예언자 님` |
| `apprentice_seer` | `견습 예언자 소녀` | `고양이 마법소녀 / 냐옹 예언자` |
| `aura_seer` | `수녀님` | `토끼 수녀님` |
| `drunk` | (유지) | `타누키 영감 / 술독 너구리` |
| `doppelganger` | `부인 / 아주머니` | `여우 부인` (키츠네 암시) |
| `cursed` | `저주받은 부인` | `저주받은 까마귀 여인` |
| `beholder` | `노학자 어르신` | `거북 선생 / 거북이 어르신` |
| `village_idiot` | (유지) | `먹보 판다` |
| `troublemaker` | `말썽쟁이 요정` | `다람쥐 요정 / 장난꾸러기 다람쥐` |
| `insomniac` | `잠 못 든 아가씨` | `잠 못 든 박쥐 아가씨` |
| `robber` | `강도 양` | `여우 강도 아가씨` |
| `tanner` | `도령` | `검은표범 도령` |

위 확장안은 **선택적**이다. TTS 대사 전량 교체 부담이 있을 경우 호칭은 기존 교정안만 적용하고, 동물화는 비주얼 층에서만 먼저 도입해도 체계가 깨지지 않는다.

## 스타일 가이드 (프롬프트 추가 키)

`role_portraits_prompts.json`의 `style_prefix` / `style_suffix`는 그대로 두고, 각 item `prompt` 머리에 아래 공용 키워드 뭉치를 덧붙여 수인 톤을 통일한다.

```
anthropomorphic beast-folk, humanoid body in Taisho-era attire, furred animal head with human expressions, hybrid yokai aesthetic, consistent with the wolf cast (werewolf/alpha_wolf/mystic_wolf/dream_wolf) in fur rendering and costume density
```

진영별 털 색 힌트는 각 프롬프트 본문에 개별로 삽입한다:
- 마을 진영: `white / cream / pale brown / pastel fur`
- 늑대 진영: `black / silver / crimson / grey fur`
- 중립·저주: `jet black / violet-black / ember-red fur`

## 리스크 · 의사결정 포인트

1. **가독성 손실**: 인간 얼굴이 사라지면 플레이어가 역할을 즉각 인식하기 어려울 수 있다. → 각 역할의 **행동 소품**(견습 예언자의 마법봉, 강도의 나무 패 두 장, 계시자의 뒤집힌 패 등)을 현재보다 더 크고 선명하게 유지해 아이콘 식별을 보완한다.
2. **tanner / thing / doppelganger 난이도**: 이 셋은 원형이 이미 "변칙적 존재"라 동물 매칭이 주관적이다. 본 제안(흑표범 / 수달+마스크 / 구미호)은 1안이며, 각 2안(`tanner`→백로, `thing`→오소리, `doppelganger`→카멜레온)을 예비로 남긴다.
3. **작업량**: 21개 초상화 재생성이 필요하다. `scripts/generate_role_portraits.py` 흐름이 이미 있으므로 `role_portraits_prompts.json` 일괄 교체 → 재생성 1회로 처리 가능한 규모다.
4. **TTS 재수록 여부**: 위 "호응" 섹션의 확장 호칭을 채택하는 경우에만 해당 역할 보이스 파일 재합성이 필요하다. 비주얼만 교체하는 경로에서는 TTS는 손대지 않는다.

## 다음 단계 제안

1. 이 문서의 캐스팅 표에 대한 승인/교체 의견 수렴.
2. 합의된 캐스팅으로 `scripts/role_portraits_prompts.json` 21개 항목의 `prompt` 필드 교체.
3. `generate_role_portraits.py` 재실행해 신규 초상 27장 일괄 재생성.
4. 늑대 진영 기존 4장과 나란히 배치해 **털 밀도·조명·프레임**이 한 계열로 보이는지 QA.
5. 필요 시 호칭 확장안(`타누키 영감`, `여우 부인` 등)을 TTS 대사에 반영하고 해당 역할만 부분 재합성.
