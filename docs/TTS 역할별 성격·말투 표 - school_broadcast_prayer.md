# TTS 역할별 성격·말투 표 - school_broadcast_prayer

이 문서는 기존 [TTS 역할별 성격·말투 표](/mnt/d/weeks/one-night-ultimate-werewolf-llm-edition/docs/TTS%20%EC%97%AD%ED%95%A0%EB%B3%84%20%EC%84%B1%EA%B2%A9%C2%B7%EB%A7%90%ED%88%AC%20%ED%91%9C.md)를 그대로 두고, `school_broadcast_prayer` 전용으로 따로 쓰는 설정표다.

목표는 세 가지다.

- `school_broadcast_prayer`의 호칭을 `학생 / 선생님 / 교직원` 축으로 다시 정렬한다.
- 학생 말투는 실제 한국 10대 구어체를 참고하되, 유행이 너무 빠른 밈이나 과한 은어는 줄이고 `교실에서 바로 들릴 법한 대화`로 맞춘다.
- 방송실, 야간 자율학습, 기도문, 복도, 교무실, 상담실, 보건실이 떠오르는 학교 호러 톤을 유지한다.

## 조사 반영 기준

온라인 자료를 먼저 확인하고 아래 원칙만 추렸다.

- 국립국어원 `모두의 말뭉치`는 일상 대화와 메신저 대화 같은 실제 언어 환경 자료를 지속적으로 공개하고 있어, 최근 한국어 구어체 감각을 잡는 기본 참고 자료로 쓰기 좋다.
- 메신저 대화 연구에서는 응답표현이 `글로 쓰지만 구어와 닮은` 특징을 보이고, 실제 카카오톡 대화에는 다양한 반응 표현과 변이형이 나타난다고 본다.
- 메신저 음운 변이 연구에서는 축약, 탈락, 장음 표기가 함께 나타나며, 일부는 타이핑을 줄이거나 실제 발화의 길이감과 강조를 옮기기 위한 방식으로 설명된다.
- 청소년 언어 연구에서는 유행어·은어·비속어 사용이 또래 집단 안에서 넓게 나타나지만, 모든 문장을 과장된 인터넷체로 만들기보다 관계와 상황에 따라 강도를 조절하는 게 더 자연스럽다.
- 청소년 언어 특성 연구는 또래 집단 안의 친밀함을 유지하려는 적응 과정이 청소년 언어 형성과 연결된다고 본다.

이 문서에서는 그 기준을 이렇게 반영한다.

- 학생 역할은 `야`, `아니`, `진짜`, `약간`, `헐`, `뭔데`, `그니까`, `맞거든`, `에바야`처럼 실제 교실 대화에 붙는 짧은 표현을 쓴다.
- 다만 한 문장을 전부 유행어로 채우지 않는다. 핵심 문장은 또렷하게 유지하고, 추임새만 학생답게 둔다.
- 선생님 역할은 `얘들아`, `지금부터`, `조용히`, `한 명씩`, `확인할게`, `돌아가`처럼 학교 현장 안내체로 쓴다.
- 교직원 역할은 `행정실`, `방송부`, `생활지도`, `야자 감독`, `당직`, `순찰`, `기록` 같은 업무 단어를 넣어 학생 말투와 분리한다.

## school_broadcast_prayer 톤 원칙

- 배경은 `방송실 스피커에서 밤 기도문이 흘러나오는 학교`다. 교실, 복도, 방송실, 상담실, 보건실, 교무실, 행정실, 당직실, 야자 자습실이 떠오르게 쓴다.
- 학생은 `친구끼리 실제로 하는 말`처럼 짧고 반응이 빠른 문장으로 간다.
- 선생님은 너무 관공서 문장처럼 굳히지 말고, 학교에서 바로 들리는 생활지도 톤으로 쓴다.
- 교직원은 학생보다는 딱딱하지만, 지나치게 문어체로 가지 않는다.
- 학교 호러는 말투보다 상황 단어로 준다. `이름 박다`, `방송 켜지다`, `복도 돌다`, `당번`, `생활기록`, `호출`, `야자`, `출석`, `지각`, `상담`, `순찰` 같은 단어가 유효하다.

## 2025 학교 말투 적용 규칙

| 규칙 | 예시 |
|------|------|
| 학생 말투는 짧게 끊는다 | `야, 걔 수상한데`, `아니 그건 좀 에반데`, `진짜 티 났거든` |
| 학생 추임새는 0~2개만 넣는다 | `헐`, `진짜`, `약간`, `아니`, `뭔데`, `그니까`, `에바야` |
| 선생님은 학생처럼 `ㅋㅋ`, `ㄹㅇ`을 직접 말하지 않는다 | `웃기지 마`는 가능하지만 `그거 ㄹㅇ이냐`는 피한다 |
| 교직원은 업무 단어를 붙여서 학생과 분리한다 | `생활지도 기록 볼게`, `방송실 순서 확인하자` |
| 유행어는 문장 전체가 아니라 꼬리만 학생답게 만든다 | `그거 수상해`보다 `그거 좀 수상한데`가 자연스럽다 |
| 호러 톤은 욕설보다 분위기로 준다 | `소름`, `등골`, `복도 끝`, `스피커`, `기도문`, `이름 불림` 같은 단어를 우선한다 |


## 성별·연령 축

이 축은 `school_broadcast_prayer` 역할별 표(하단)와 [비주얼 매칭 표](TTS%20%EC%97%AD%ED%95%A0%EB%B3%84%20%EB%B9%84%EC%A3%BC%EC%96%BC%C2%B7%EB%B3%B4%EC%9D%B4%EC%8A%A4%20%EB%A7%A4%EC%B9%AD%20%ED%91%9C.md)를 교차 반영해 학교 현장 기준으로 다시 짠 것이다. 학생 역할은 모두 10대(소년/소녀)로 내리고, 교사·교직원은 담당 업무와 비주얼에 맞춘 중장년 톤으로 맞춘다.

| 축 | 범위 | 대표 역할 | 학교 현장 매칭 |
|------|------|------|------|
| 중장년 남성 | 50~60대, 묵직한 위압, 규율형 저음 | `alpha_wolf`, `beholder` | 학생부장, 야자 감독 / 교무부 원로 |
| 중년 남성 | 40~50대, 낮은 저음, 생활지도 톤 | `werewolf`, `minion` | 담임, 부담임 |
| 청장년 남성 | 30대, 건조·거친 톤, 마스크 이미지 | `thing` | 복도에서 마주치는 낯선 어른 |
| 중년 여성 | 40대 전후, 차분·또렷, 방송 진행형 | `seer` | 방송실 흐름 읽는 지도쌤 |
| 청장년 여성 | 30대, 관능/광기/이중성 | `mystic_wolf`, `witch` | 상담쌤, 방송실 담당쌤 |
| 노년 여성 | 할머니 톤, 나른한 저음, 꿈꾸는 듯한 장음 | `dream_wolf` | 오래 자리 지킨 보건쌤 |
| 청년 여성 | 20대, 사무·신입·또렷 | `paranormal_investigator`, `revealer` | 신임 생활지도·교무쌤 |
| 소년 | 10대 후반 남학생, 변성기 직후, 반말·추임새 | `squire`, `mason`, `village_idiot`, `drunk` | 총무, 방송부 선배, 얼간이, 취기 학생 |
| 소녀 | 10대 후반 여학생, 또래 대화, 빠른 반응 | `doppelganger`, `apprentice_seer`, `insomniac`, `robber`, `aura_seer`, `troublemaker` | 전학생, 방송부 막내, 불면증, 매점 뒤 애, 눈치 빠른 애, 말썽쟁이 |

> 참고: Narrator는 야간 방송부 담당쌤·당직 교사 톤이라 위 축에 굳이 묶지 않는다. 상황에 맞춰 중년 남성 또는 중년 여성 성우 중 고른다.

## 빠른 작성 팁

| 상황 | 권장 방식 |
|------|------|
| 학생 역할이 전부 인터넷 밈처럼 들릴 때 | `헐`, `진짜`, `약간`, `그니까`, `에바` 정도만 넣고 핵심 설명은 평범한 한국어로 둔다. |
| 학생과 선생님 톤이 안 갈릴 때 | 학생은 반말, 선생님은 `얘들아 / 조용히 / 확인해 볼게요 / 돌아가세요` 같은 생활지도 말투로 고정한다. |
| 학교 느낌이 약할 때 | `방송실`, `기도문`, `야자`, `생활지도`, `상담실`, `보건실`, `교무실`, `당직` 단어를 넣는다. |
| 호러가 너무 약할 때 | `복도 끝`, `스피커`, `이름 불림`, `찢긴 종이`, `밤 방송`, `불 꺼진 교실` 같은 장면 단어를 앞문장에 둔다. |
| 유행어가 너무 많을 때 | 한 클립당 학생 추임새 1~2개면 충분하다. 나머지는 자연스러운 교실 대화로 유지한다. |
| 선생님 말투가 너무 딱딱할 때 | `확인 바랍니다`보다 `확인해 볼게요`, `지금은 조용히 해 주세요` 쪽이 학교 현장에 더 가깝다. |

## 권장 템플릿

```text
[학교 분위기 1문장]. [학생/선생님/교직원 호칭], [눈 떠 표현]. [행동 핵심]. [필요하면 학생 추임새나 학교식 안내 표현 0~1회]. [눈 감기 표현].
```

예시 1 (학생 톤):

```text
복도 끝 스피커에서 잡음이 한 번 튀었어. 도플갱어 전학생, 이제 눈 떠 봐. 한 명만 보고 그 역할 따라 해 봐. 헐, 말투 하나만 비슷해도 진짜 잘 안 들키거든. 티 나기 전에 얼른 감아.
```

예시 2 (선생님 톤):

```text
방송실 불은 꺼졌는데 믹서만 켜져 있네요. 상담쌤, 눈 떠 보세요. 다른 플레이어 한 명만 차분히 확인해 보세요. 반응은 생각보다 얼굴에 먼저 뜨거든요. 본 건 숨기고 다시 감아 주세요.
```

예시 3 (교무실 톤):

```text
야자 종료 방송 전에 확인할 게 있겠습니다. 생활지도 심령 수사관 선생님, 눈 떠 보겠습니다. 다른 플레이어 카드를 최대 두 장까지 순서대로 확인해 보겠습니다. 멈춘 지점은 그대로 기록합니다. 여기까지 보고 다시 감겠습니다.
```
