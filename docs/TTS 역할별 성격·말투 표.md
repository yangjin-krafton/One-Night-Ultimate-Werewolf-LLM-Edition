# TTS 역할별 성격·말투 표

이 문서는 `roleClips`를 쓸 때 역할마다 말투가 겹치지 않도록 기준을 잡기 위한 표다.

특히 아래 5가지를 역할별로 분리해서 쓴다.

- 역할을 부를 때 쓰는 호칭과 호출 앞뒤 표현
- `눈 떠 주세요`를 대체하는 개성 있는 표현
- `눈 감아 주세요`를 대체하는 개성 있는 표현
- `음...`, `아하`, `후후`, `킥킥` 같은 추임새
- 문장 마무리 어미와 말끝 습관

## 공통 원칙

- 한 역할은 `항상 같은 말투 축`을 유지한다.
- 한 클립에서 개성 포인트를 전부 다 쓰지 않는다. `호칭 1개 + 눈 떠 표현 1개 + 눈 감기 표현 1개 + 추임새 0~1개` 정도면 충분하다.
- 겹치는 캐릭터가 생기지 않게 팀과 기능에 따라 어조를 분리한다.
- 모든 역할을 억지로 `-세요`로 끝내지 않는다. 역할에 따라 `-해`, `-하지`, `-해라`, `-하면 돼`, `-하시죠`, `-하시면 됩니다`처럼 종결을 갈라 쓴다.
- `ep1`은 설명을 길게 하되 말투 축은 유지한다.
- `ep2`는 설명을 줄이고 표현만 남긴다.
- `ep3`는 가장 짧게 줄이되, 역할의 호칭과 어미만으로도 누군지 느껴지게 한다.

## 조사 반영 기준

2025년 기준으로 바로 참고 가능한 한국어 대사/자막 계열 자료는 아래 순서로 반영했다.

- 국립국어원 `2023 일상 대화 말뭉치`와 `모두의 말뭉치 분석기`: 실제 한국어 구어에서 어떤 종결어미가 자주 쓰이는지 보는 1차 기준
- AI Hub `한국어 음성`, `회의 음성`: 머뭇거림, 간투사, 자동 자막 생성용 전사 데이터 확인용
- OPUS `OpenSubtitles`: 영화/드라마 자막 계열의 빠른 호흡과 짧은 대사 리듬 참고용

이 문서에서 실제로 반영한 원칙은 아래와 같다.

- 기본 구어체의 중심은 `-해`, `-하지`, `-야`, `-지`, `-거든`, `-네`, `-해요`, `-죠`, `-하시죠`, `-합시다`, `-시면 됩니다` 쪽으로 둔다.
- `-세요`는 진행자나 안내형 역할에만 제한적으로 쓴다.
- 자막 말투처럼 짧고 빨리 지나가는 줄은 `보시죠`, `가면 돼요`, `돌아가`, `지금은 자`, `여기까지죠`처럼 짧은 종결을 우선한다.
- 역할 하나 안에서는 존댓말 축과 반말 축을 섞지 않는다. 섞어야 할 경우에도 `기본 종결 1개 + 변주 1개` 수준으로만 쓴다.
- 온라인 자막은 품질 편차가 커서 그대로 베끼지 않고, `짧은 문장 길이`와 `자연스러운 종결 패턴`만 참고한다.
- 호칭은 문어체 수식보다 `님`, `아/야`, `대장`, `선생`, `반장`, `전하`처럼 실제로 사람을 부를 때 입에 붙는 짧은 말로 잡는다.
- 호칭 앞뒤 표현은 `이 밤의`, `어둠을 읽는` 같은 서문체 대신 `다음은`, `이번엔`, `너`, `님`, `친구`, `담당`, `쪽`처럼 대화형 표현을 우선한다.

## 말끝 축

| 말끝 축 | 실제 대화에서의 느낌 | 권장 역할 |
|------|------|------|
| `-해 / -하지 / -지 / -거든` | 가까운 사이의 자연스러운 반말, 능청, 투덜거림 | 강도, 말썽쟁이, 불면증환자, 도플갱어 |
| `-해라 / -하라 / -다` | 명령, 위압, 사냥감 몰이 | 늑대인간, 태초의 늑대인간, 사냥꾼 |
| `-해요 / -봐요 / -볼까요 / -잖아요` | 부드럽고 살짝 거리 있는 구어 존댓말 | 마녀, 신비한 늑대인간, 영기 예언자 |
| `-하시죠 / -둡시다 / -입니다` | 설명형, 단정형, 브리핑 톤 | 예언자, 심령 수사관, 계시자 |
| `-하시면 됩니다 / -하십시오` | 절차 안내, 보좌, 의식 진행 | 진행자, 종자 |
| `-하면 돼 / -있어 / -지 마` | 체념, 혼잣말, 수동 상태 | 잠자는 늑대인간, 무두장이, 견습 무두장이 |

## 호칭 축

| 호칭 축 | 온라인/대화체에서의 느낌 | 권장 역할 |
|------|------|------|
| `님` | 가볍게 존중하면서도 딱딱하지 않은 호칭 | 예언자, 마녀, 주시자, 계시자 |
| `아/야` | 친한 사이, 장난기, 빠른 호출 | 강도, 말썽쟁이, 견습 예언자, 도플갱어 |
| 직함형 | 역할을 사람처럼 부르는 방식 | 대장, 수사관님, 왕자님, 저격수 |
| 일반명사형 | `친구`, `사람`, `쪽`, `담당`처럼 부드럽게 돌려 부르는 방식 | 주민, 영기 예언자, 저주받은 사람 |
| 별명형 | 놀림, 애칭, 캐릭터 강조 | 잠꾸러기, 사고뭉치, 불면러, 술꾼님 |

## 역할군 분리 기준

| 역할군 | 톤 기준 |
|------|------|
| 진행자 | 의식 진행, 리듬 제어, 또렷한 선언형 |
| 늑대 팀 | 포식, 유혹, 냉소, 비밀스러운 압박 |
| 정보 확인형 | 차분, 관찰, 분석, 낮은 감탄 |
| 교란/교환형 | 장난, 민첩, 즉흥, 가벼운 도발 |
| 수동/특수 승리형 | 자기 합리화, 체념, 버팀, 기묘한 확신 |

## 빠른 작성 팁

| 상황 | 권장 방식 |
|------|------|
| 같은 팀인데 말투가 겹칠 때 | 늑대인간은 `명령형`, 신비한 늑대인간은 `유혹형`, 하수인은 `비굴한 충성형`, 종자는 `공손한 보좌형`으로 나눈다. |
| 정보직이 모두 차분해서 비슷할 때 | 예언자는 `계시`, 견습 예언자는 `조심`, 심령 수사관은 `조사`, 영기 예언자는 `감지`, 주시자는 `보호` 어휘를 고정한다. |
| 장난형 역할이 모두 산만할 때 | 강도는 `능청`, 말썽쟁이는 `장난`, 동네 얼간이는 `엉뚱`, 주정뱅이는 `취기`로 구분한다. |
| 밤 행동이 없는 역할이 심심할 때 | 행동 대신 `태도`를 준다. 사냥꾼은 `대기`, 무두장이는 `죽음 집착`, 왕자는 `여유`, 주민은 `담백함`, 저주받은 사람은 `불안`으로 간다. |
| 한국어가 부자연스럽게 딱딱할 때 | `확인하세요` 대신 `한번 봐`, `살펴봐요`, `확인하시죠`, `가만히 있으면 돼요`, `돌아가`, `덮어 둬`처럼 실제 대화에서 나올 법한 종결로 바꾼다. |
| 자막 말투처럼 짧게 쳐야 할 때 | 한 줄에 핵심 동사 하나만 세운다. `봐`, `골라`, `돌아가`, `숨겨`, `버텨`, `넘겨요`, `기다리시죠` 정도가 가장 안정적이다. |

## 권장 템플릿

```text
[호출 분위기 1문장]. [역할명], [눈 떠 표현]. [행동 핵심]. [필요하면 추임새/주의 1문장]. [눈 감기 표현].
```

예시:

```text
솥 안이 슬슬 끓네요. 마녀, 눈 떠 봐요. 중앙 카드 한 장 보고 마음에 들면 그걸로 운명을 바꿔도 돼요. 후후, 티 나기 전에 주문 접고 돌아가요.
```
