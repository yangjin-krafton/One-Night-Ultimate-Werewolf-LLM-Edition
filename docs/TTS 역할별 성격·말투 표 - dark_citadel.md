# TTS 역할별 성격·말투 표 - dark_citadel

이 문서는 기존 [TTS 역할별 성격·말투 표](/mnt/d/weeks/one-night-ultimate-werewolf-llm-edition/docs/TTS%20%EC%97%AD%ED%95%A0%EB%B3%84%20%EC%84%B1%EA%B2%A9%C2%B7%EB%A7%90%ED%88%AC%20%ED%91%9C.md)를 건드리지 않고, `dark_citadel` 전용으로 따로 쓰는 설정표다.

목표는 네 가지다.

- `dark_citadel`의 호칭을 한국식 밈 톤보다 `중세 유럽 성채 / 왕실 / 귀족 / 기사단 / 시민` 톤으로 재정렬한다.
- 역할 이름은 게임 규칙용 원래 이름을 유지하되, 실제 호출문은 `폐하`, `전하`, `공작 각하`, `기사`, `영주`, `시민`, `수녀`, `학자` 같은 계층형 호칭으로 바꾼다.
- 추임새는 `럭셔리`, `르네상스`, `센스`, `해피`처럼 한국에서 입말처럼 소비되는 콩글리시를 역할별 시그니처로 배치한다.
- **전체 대사의 리듬은 해외 영화 한국어 더빙판 특유의 어색한 번역체로 고정한다.** 어휘는 한국어지만 문장 구조와 감탄사가 외화 더빙처럼 살짝 비틀려 있어야 한다.

## dark_citadel 톤 원칙

- 배경은 `저주받은 중세 유럽 성채`다. 궁정, 기사단, 수도원, 장터, 지하 감옥, 망루 같은 공간을 떠올리게 쓴다.
- 호칭은 `이 밤의`, `어둠의 화신` 같은 과한 서문체보다 실제 사람을 부르는 짧은 호명으로 쓴다.
- 같은 역할이라도 `ep1/ep2/ep3`에서 설명량은 달라져도, 호칭 계층과 말끝 축은 유지한다.
- 콩글리시는 장식처럼 쓴다. 한 클립에 0~1회가 적당하다.
- `럭셔리`, `르네상스`, `센스`, `해피`는 한국식 입말 감탄처럼 쓰고, 영문 원뜻 설명처럼 쓰지 않는다.
- 중세 유럽풍이라고 해서 한국어가 문어체로 굳으면 안 된다. 말투는 살아 있고, 귀족만 격식을 갖춘다.

## 해외 영화 한국어 더빙판 톤 원칙

`dark_citadel`은 단순 판타지 톤이 아니라, **1980~1990년대 지상파에서 틀어 주던 외화 더빙판** 특유의 어색한 번역체를 일부러 껴안는다. 실제 모국어 화자가 쓰는 자연스러운 한국어가 아니라, `원문 어순을 억지로 살린 한국어`라는 점이 핵심이다.

- 주어를 자주 명시한다. 한국어는 주어를 생략해도 되지만, 더빙판은 `당신은`, `나는`, `우리는`을 붙인다. 예: `당신은 지금 왕좌에 앉아 있는 거요`.
- 감탄사를 직역해서 쓴다. 예: `오, 이런`, `맙소사`, `세상에`, `저런, 저런`, `오 주여`, `빌어먹을`, `제기랄`.
- 호칭을 문장 끝에 붙인다. 예: `조심하시오, 친구여`, `서두르게, 경`, `그렇군요, 폐하`.
- 부자연스러운 격식체를 섞는다. 예: `~하는 것이 좋을 것 같소`, `~라는 말이지요?`, `~하지 않을 수 없군`, `이 얼마나 곤란한 일인가`.
- 장음을 과장한다. 예: `오오오`, `세에상에`, `마아안소사`. 한두 군데에만 배치해서 리듬만 살린다.
- `당신` 남용은 의도적이다. 한국어에서 어색하지만 외화 더빙에서는 빈번하다. 예: `당신이 본 것은 절대 거짓이 아니오`.
- 이름·호칭 반복을 활용한다. 예: `기사여, 기사여, 내 말을 들으시오`. 긴박함을 강조할 때 한 번만.
- 형용사를 문장 뒤에 붙이는 `도치형` 번역투도 쓴다. 예: `이건 정말이지 수상한 일이군요, 정말로`.
- 욕설·거친 표현은 외국식 직역을 쓴다. 예: `빌어먹을`, `제기랄`, `망할`, `이런 젠장`. 늑대 팀, 도둑 같은 거친 역할에만.
- 신파조·감상형 번역도 허용한다. 예: `오, 이 얼마나 비극적인가`, `운명이란 참으로 가혹하군요`. 무두장이, 저주받은 자에게만.

## 더빙 번역투 축

콩글리시와 별개로, 역할마다 주력 `번역투 리듬`을 하나씩 고정한다. 한 클립에 번역투 어구 1~2개면 충분하다.

| 번역투 표현 | 쓰는 느낌 | 권장 역할 |
|------|------|------|
| `이봐, 친구 / 이봐, 자네` | 친근한 서양식 호명 직역 | robber, troublemaker, village_idiot, insomniac |
| `오, 이런 / 세에상에 / 맙소사` | 놀람·당혹의 직역 감탄사 | dream_wolf, drunk, cursed, insomniac |
| `당신은 ~하는군요 / 당신은 ~이오` | 주어 강조 직역체 | seer, paranormal_investigator, revealer |
| `~라는 말이지요? / ~라는 뜻인가` | 확인·재진술 번역투 | paranormal_investigator, beholder, aura_seer |
| `~하는 것이 좋을 것 같소 / ~하시는 편이 현명하오` | 격식 더빙, 노년 귀족풍 | werewolf, alpha_wolf, prince |
| `빌어먹을 / 제기랄 / 이런 젠장` | 거친 외국 감탄 직역 | werewolf, robber, minion |
| `저런, 저런 / 아이고, 아이고` | 동정 감탄 반복 | villager, dream_wolf |

| `내 말을 들으시오 / 자네, 내 말을` | 호소형 직역 | squire, beholder, hunter |
| `후후, ~하는 것이지요` | 악당 여유 번역투 | witch, mystic_wolf |

## 빠른 작성 팁

| 상황 | 권장 방식 |
|------|------|
| 늑대 팀이 다 비슷해질 때 | `werewolf`는 군주, `alpha_wolf`는 대왕, `mystic_wolf`는 공작, `dream_wolf`는 잠든 왕자로 나눠서 계층부터 다르게 잡는다. |
| 예언 계열이 전부 학자처럼 겹칠 때 | `seer`는 궁정 예언관, `apprentice_seer`는 견습 서기관, `aura_seer`는 점성관, `paranormal_investigator`는 심문관으로 갈라 쓴다. |
| 시민층이 너무 가벼워질 때 | `robber`는 골목 도둑, `drunk`는 선술집 주인장, `insomniac`는 야경꾼, `village_idiot`는 시장통 사고뭉치로 고정한다. |
| 콩글리시가 너무 많아질 때 | 한 줄에 하나만 쓴다. `럭셔리하네`, `르네상스네`, `센스 있네`, `해피는 아니네` 정도면 충분하다. |
| 중세풍인데 너무 문어체가 될 때 | 호칭만 유럽풍으로 하고, 동사는 `봐`, `고르시죠`, `돌아가요`, `감아`, `물러나소서`처럼 입에 붙는 짧은 한국어로 유지한다. |
| 더빙 느낌이 약할 때 | 주어 `당신은 / 그대는`을 일부러 한 번 넣고, `오, 이런`이나 `맙소사` 같은 직역 감탄사를 문장 맨 앞에 꽂는다. |
| 더빙 느낌이 너무 과할 때 | 한 클립에 번역투 어구 2개까지만. 그 이상이면 놀림처럼 들려 몰입이 깨진다. |
| 신파 번역투가 과도할 때 | `오, 이 얼마나 ~인가` 같은 구문은 `tanner`, `cursed`, `apprentice_tanner`에만 쓰고 다른 역할에는 쓰지 않는다. |
| 욕설 번역투가 튈 때 | `빌어먹을 / 제기랄` 같은 직역 욕은 늑대 팀, 도둑, 사냥꾼에만 허용하고 왕실·학자·기사에는 쓰지 않는다. |

## 권장 템플릿

```text
[성채 분위기 1문장]. [유럽풍 호칭], [눈 떠 표현]. [행동 핵심]. [더빙 번역투 + 콩글리시 0~1회]. [눈 감기 표현].
```

예시 1 (궁정 귀족, 공작 각하):

```text
성채 복도 공기가 오늘따라 로열하네. 공작 각하, 이제 눈 들어 보시지요. 다른 플레이어 한 명만 조용히 읽어 두시지요. 후후, 당신이 감추려 해도 나에겐 보이지요, 르네상스네. 비밀은 접어 두고 물러나시지요.
```

예시 2 (심문관, 셜록 홈즈 풍 더빙):

```text
이 밤의 공기는 뭔가 수상하군요. 심문관님, 이제 눈 뜨시죠. 다른 플레이어 카드를 최대 두 장까지 한 장씩 보시면 됩니다. 그러니까, 당신의 말은 전부 의심스럽다는 뜻이지요? 센스 있네. 기록만 챙기고 물러나시면 됩니다.
```

예시 3 (왕의 폐하, 과장된 군주 더빙):

```text
왕좌 위로 피 냄새가 번지는군. 늑대 폐하, 이제 눈 드소서. 다른 늑대가 있다면 서로 얼굴을 확인하시고, 혼자시라면 중앙 카드 한 장을 들춰 보시지요. 빌어먹을 인간들, 짐의 왕좌를 넘보다니 이 얼마나 어리석은가. 이제 다시 어둠으로 드시지요.
```

예시 4 (선술집 주인장, 잭 스패로 풍 더빙):

```text
선술집 바닥이 오늘도 빙글빙글 도는군요오. 주인장, 비틀비틀 깨 보자아. 자기 카드랑 중앙 카드 한 장을 바꾸는 거야아, 근데 새 카드는 절대 못 봐요오. 이봐, 친구이이... 마아안소사, 이게 무슨 운명인가. 어지러우니까 얼른 감아아.
```
