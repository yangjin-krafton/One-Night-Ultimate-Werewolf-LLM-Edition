# TTS 역할별 성격·말투 표 - salgol_ward

이 문서는 기존 [TTS 역할별 성격·말투 표](/mnt/d/weeks/one-night-ultimate-werewolf-llm-edition/docs/TTS%20%EC%97%AD%ED%95%A0%EB%B3%84%20%EC%84%B1%EA%B2%A9%C2%B7%EB%A7%90%ED%88%AC%20%ED%91%9C.md)를 그대로 두고, `salgol_ward` 전용으로 따로 쓰는 설정표다.

목표는 세 가지다.

- `salgol_ward`의 호칭을 실제 한국 병원 현장에서 들을 법한 `의료진 / 병원 지원 / 환자` 톤으로 재정렬한다.
- 의료진 역할은 `선생님`, `간호사`, `전공의`, `교수`, `수간호사`, `원무`, `보호사`처럼 병원에서 실제로 입에 붙는 호칭으로 쓴다.
- 환자와 보호자 톤은 과장된 호러 문어체보다, 한국 병동에서 흔히 들리는 설명형·안심형 말투를 바탕으로 한다.

## 조사 반영 기준

온라인 자료를 먼저 확인하고 아래 원칙만 추렸다.

- 환자는 병원에서 자신을 `OOO님`으로 불러 주는 걸 가장 선호했지만, 실제 의료진은 `환자분`을 더 많이 쓰는 경향이 있다.
- 병원 현장에서는 직군을 넘어서 `선생님`이 넓게 쓰이고, 일부 병원은 직원 호칭을 `선생님`으로 통일하기도 한다.
- 간호·응급실 커뮤니케이션 연구에서는 검사·치료 진행 설명, 정보 제공, 현재 상태 안내가 중요하게 다뤄진다.

이 문서에서는 그 기준을 이렇게 반영한다.

- 환자를 직접 부를 때는 `OOO님`, `환자분`, `보호자분`을 우선한다.
- 의료진끼리 또는 역할 호출 시에는 `선생님`, `간호사`, `전공의`, `교수님`, `수간호사님`을 우선한다.
- 병동 말투는 `지금 확인할게요`, `조금만 기다려 주세요`, `기록 먼저 볼게요`, `지금 상태만 보시면 됩니다`처럼 짧고 안내형으로 간다.
- 폐쇄병동 호러 톤은 `차트`, `병실`, `회진`, `오더`, `인계`, `격리`, `기록`, `보호자`, `진정`, `야간근무` 같은 단어로 준다.

## salgol_ward 톤 원칙

- 배경은 `폐쇄병동 야간 인수인계`다. 병실, 스테이션, 처치실, 보호실, 원무창구, 휴게실, 차트실이 떠오르게 쓴다.
- 병원 특유의 직업 호칭은 유지하되, 플레이어가 듣기 어렵지 않게 너무 깊은 전문용어는 남발하지 않는다.
- 의료진은 과장된 감정보다 `설명`, `관찰`, `기록`, `안심`, `지시`를 우선한다.
- 병원 지원 인력은 현장 실무 톤, 환자 계열은 불안·예민·억울함·체념 쪽으로 차이를 준다.
- 같은 역할이라도 `ep1/ep2/ep3`에서 설명량은 줄어들 수 있지만, 직군 호칭과 말끝 축은 유지한다.

## 성별·연령 축

이 축은 `salgol_ward` 역할별 표(하단)와 [비주얼 매칭 표](TTS%20%EC%97%AD%ED%95%A0%EB%B3%84%20%EB%B9%84%EC%A3%BC%EC%96%BC%C2%B7%EB%B3%B4%EC%9D%B4%EC%8A%A4%20%EB%A7%A4%EC%B9%AD%20%ED%91%9C.md)를 교차 반영해 폐쇄병동 기준으로 다시 짠 것이다. 의료진은 담당 직책과 비주얼에 맞게 중장년 톤으로, 환자·지원 인력은 연령대를 장면에 맞게 분산한다.

## 빠른 작성 팁

| 상황 | 권장 방식 |
|------|------|
| 환자 호칭이 너무 거칠 때 | 직접 부를 땐 `OOO님`, `환자분`, `보호자분`을 우선한다. |
| 의료진 역할이 전부 의사처럼만 들릴 때 | `교수님`, `전공의`, `수간호사`, `간호사`, `원무과`, `보호사`로 분리한다. |
| 병원 느낌이 약할 때 | `차트`, `병실`, `회진`, `오더`, `인계`, `격리`, `기록`, `보호자` 단어를 넣는다. |
| 호러 톤 때문에 말투가 과장될 때 | 공포는 상황과 단어로 주고, 말투는 실제 병동처럼 차분하게 둔다. |
| 환자/보호자와 의료진 말투가 안 갈릴 때 | 의료진은 설명형 존댓말, 실무자는 짧은 반말/실무체, 환자는 불안/체념 섞인 구어체로 나눈다. |

## 권장 템플릿

```text
[병동 분위기 1문장]. [병원식 호칭], [눈 떠 표현]. [행동 핵심]. [필요하면 의료 현장 추임새 0~1회]. [눈 감기 표현].
```

예시:

```text
야간 스테이션 공기가 오늘따라 이상하네요. 예언자 교수님, 눈 떠 보시죠. 한 사람의 차트를 읽거나 중앙 두 장을 확인해 보시죠. 기록상 안 맞는 쪽은 나중에 바로 드러납니다. 차트만 닫고 다시 감으시죠.
```
