# TTS 역할별 성격·말투 표 - floodgate_nameplates

이 문서는 기존 [TTS 역할별 성격·말투 표](/mnt/d/weeks/one-night-ultimate-werewolf-llm-edition/docs/TTS%20%EC%97%AD%ED%95%A0%EB%B3%84%20%EC%84%B1%EA%B2%A9%C2%B7%EB%A7%90%ED%88%AC%20%ED%91%9C.md)를 그대로 두고, `floodgate_nameplates` 전용으로 따로 쓰는 설정표다.

목표는 세 가지다.

- `floodgate_nameplates`의 호칭을 조선 시대 `왕실 / 사또 / 양반 / 포도청 / 궁중 / 백성 / 요물` 톤으로 재정렬한다.
- 역할 이름은 게임 규칙용 원래 이름을 유지하되, 실제 호출문은 `전하`, `대감`, `나리`, `영감`, `포졸`, `아씨`, `주모`, `백성` 같은 조선식 호칭으로 바꾼다.
- 대사 말투는 사극풍이되 완전 문어체로 굳히지 않고, 입에 붙는 한국어로 정리한다.

## floodgate_nameplates 톤 원칙

- 배경은 `비가 잦은 조선 수문 관아`다. 수문, 관아 마당, 포도청, 사랑채, 초가, 우물가, 서고, 장터 같은 공간을 떠올리게 쓴다.
- 호칭은 `어둠의 사자`, `비밀의 화신` 같은 추상적 서문체보다 `대감`, `나리`, `영감`, `아씨`, `주모`, `포졸`처럼 사람이 실제로 부를 법한 말로 쓴다.
- 높은 신분일수록 `-시지요`, `-옵니다`, `-하시오`, `-하라`를 쓰고, 백성이나 장터 인물은 `-해`, `-하우`, `-하쇼`, `-거든` 같은 입말을 쓴다.
- `ep1/ep2/ep3`에서 설명량은 달라져도, 조선 계층 호칭과 말끝 축은 유지한다.
- 조선풍이라고 해서 무조건 고어투로 밀지 않는다. 플레이어가 바로 들을 수 있는 자연스러운 현대 한국어 기반 사극 톤으로 유지한다.

## 빠른 작성 팁

| 상황 | 권장 방식 |
|------|------|
| 왕실 역할이 다 비슷해질 때 | `werewolf`는 대군, `alpha_wolf`는 상왕, `dream_wolf`는 잠든 왕손으로 나눠서 높낮이부터 갈라 쓴다. |
| 조정 인물들이 모두 대감처럼만 들릴 때 | `mystic_wolf`는 좌상, `paranormal_investigator`는 포도청 수사관, `revealer`는 공개 심문관, `seer`는 궁중 예언관으로 갈라 쓴다. |
| 장터 인물들이 너무 가벼워질 때 | `drunk`는 주모/술꾼, `village_idiot`는 장터 사고뭉치, `insomniac`는 야경꾼, `robber`는 장물아비로 고정한다. |
| 사극풍이 너무 과해서 알아듣기 어려울 때 | 호칭만 조선식으로 두고 동사는 `봐`, `고르시오`, `돌아가`, `감으시지요`, `기억해 둬`처럼 짧고 익숙한 표현으로 유지한다. |
| 한 줄이 너무 장황할 때 | 조선풍 수식은 첫 문장에만 넣고, 행동 설명은 규칙 위주로 짧게 간다. |

## 권장 템플릿

```text
[관아/수문 분위기 1문장]. [조선식 호칭], [눈 떠 표현]. [행동 핵심]. [필요하면 사극풍 추임새 0~1회]. [눈 감기 표현].
```

예시:

```text
수문 밖 빗소리가 심상치 않구려. 예언관 나리, 이제 눈 들어 보시지요. 한 사람의 이름패를 읽거나 중앙 두 장을 보시지요. 통촉하여, 출처부터 캐는 입은 대개 제 속이 급한 법입니다. 계시는 접어 두시고 다시 물러나시지요.
```
